File Requirements for Web Images:

  • Format: PNG, JPG, GIF, SVG, or WebP
  • Color Mode: RGB (not CMYK)
  • Resolution: 72-150 DPI is fine for web
  • File size: Under 1MB for logos
  • Transparency: PNG and SVG support transparency
  • Path: Must be exact (case-sensitive on some servers)

Debugging Steps:

  1. Open Browser DevTools (F12)
  2. Go to Network tab
  3. Refresh this page
  4. Look for logo.png in the list
  5. Check the Status Code (should be 200)
  6. If it's 404, the file isn't in the right place
